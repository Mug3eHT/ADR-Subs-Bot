# 🤖 Инструкция по запуску бота на Railway

## Шаг 1 — Создай бота в Telegram
1. Напиши @BotFather в Telegram
2. Отправь /newbot
3. Придумай имя и username боту
4. Скопируй токен (выглядит как 1234567890:AAF...)

## Шаг 2 — Узнай Telegram ID байеров
Каждый байер пишет боту @userinfobot
Бот ответит их ID (число типа 123456789)

## Шаг 3 — Вставь данные в bot.py
Открой файл bot.py и замени:
- BOT_TOKEN — вставь токен от BotFather
- telegram_id: 111111111 — вставь ID Ивана
- telegram_id: 222222222 — вставь ID Камиля

## Шаг 4 — Задеплой на Railway
1. Зайди на railway.app и зарегистрируйся
2. Нажми "New Project" → "Deploy from GitHub repo"
   ИЛИ нажми "New Project" → "Empty project" → загрузи файлы вручную
3. Загрузи три файла: bot.py, requirements.txt, Procfile
4. Railway автоматически запустит бота
5. Во вкладке "Settings" → "Domains" — нажми "Generate Domain"
6. Скопируй свой URL (например https://bot-production-xxxx.up.railway.app)

## Шаг 5 — Настрой постбэк в Scaleo
В настройках постбэка Scaleo вставь URL:

https://ТВОЙ_URL.up.railway.app/postback?campaign={sub_id1}&status={conv_status}

## Шаг 6 — Байеры пишут боту /start
Каждый байер должен написать боту /start один раз
чтобы бот мог отправлять им сообщения.

## Шаг 7 — Проверка
Открой в браузере:
https://ТВОЙ_URL.up.railway.app/postback?campaign=Ivan_Test&status=approved

Иван должен получить уведомление в Telegram ✅

---

## Как это работает
- Байер называет кампанию с ключевым словом: Ivan_Campaign или Campaign_Kamil
- Scaleo при каждой конверсии (подписчике) дёргает постбэк URL
- Бот находит нужного байера по ключевому слову в названии кампании
- Отправляет уведомление в личку байеру
